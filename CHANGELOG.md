# Changelog

All notable changes to the CPS230 Solution will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-03-20

### Added
- Initial release of CPS230 Solution
- Process management with BPMN canvas
- Azure AD SSO and local authentication
- Row-level security in PostgreSQL
- Role-based access control (user, business_analyst, promaster)
- Azure Functions backend with 17 API endpoints
- React frontend with dark mode support
- Bicep infrastructure-as-code
- Automated deployment scripts (Bash, PowerShell)
- Customer update script with migration tracking
- Database migrations framework (001-009)
- Application Insights monitoring
- Key Vault secrets management
- Cost-optimized deployment option (~$35-55/month)
